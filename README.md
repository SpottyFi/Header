## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

### Running the Program

```sh
  npm run seed
  npm run dev
  npm run test
```

### Making bundle.js

```sh
  npm run 
  npm start
```
